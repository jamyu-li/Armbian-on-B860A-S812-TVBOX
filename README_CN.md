  用它创建属于 ZTE ZXV10 B860A (晶晨 S812芯片) 机顶盒的 Armbian Linux.
  ZTE ZXV10 B860A 系列机顶盒在中国非常普遍，多采用Amlogic S9XXX SoC。B860A是早期的一个型号，采用的是Amlogic S812 32位SoC，1GB内存，4GB EMMC，2 USB，1 TF卡，安卓4.4.2系统。这个项目的目的是创建一个能在B860A上运行的 Armbian Linux。为此我写了一些补丁脚本以及设备树，将它们运用在 Armbian Build中编译属于B860A的Linux 主线内核。
  也可以使用我编译好的镜像直接从SD卡启动ARmbian.
